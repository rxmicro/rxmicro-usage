=== Project Metadata Configuration

Для получения метаданных в процессе построения проектной документации, фреймворк RxMicro может использовать:

* Файл `pom.xml`;
* Аннотации модуля `rxmicro.documentation`.

Метаданными для проектной документации программного обеспечения являются:

* название программного обеспечения;
* описание программного обеспечения;
* версия программного обеспечения;
* список лиценций, под которыми распростаняется данное программное обеспечение;
* список разработчиков программного обеспечения;
* базовый endpoint для всех REST-based микросервисов.
