=== Integration with `rxmicro.validation` Module

Модуль `rxmicro.documentation.asciidoctor` интегрирован с модулем `rxmicro.validation`.
Благодаря данной интеграции при построении проектной документации анализируются все доступные built-in constraints для автоматической генерации описания полей модели.
