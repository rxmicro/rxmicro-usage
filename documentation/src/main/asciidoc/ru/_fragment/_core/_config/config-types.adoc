==== Типы конфигурирования

The RxMicro framework поддерживает следующие типы конфигурирования:

. Конфигурирование с помощью `classpath` ресурсов;
. Конфигурирование с помощью `properties` файлов;
. Конфигурирование с помощью переменных окружения;
. Конфигурирование с помощью Java системных переменных;
. Конфигурирование с помощью Java классов;
. Конфигурирование с помощью аннотаций;
