==== Using the Debug Mode

To get a better idea of how the RxMicro framework works, You can use the debug mode.

To do this, set breakpoints and start the microservice in debug mode.

.Starting the microservice in debug mode.
image::core/how-it-work/micro-service-start-debug.jpg[]

[NOTE]
====
The code generated by the `RxMicro Annotation Processor` is a usual source code.
So You can use breakpoints for this code as well as for Your source code.
====
