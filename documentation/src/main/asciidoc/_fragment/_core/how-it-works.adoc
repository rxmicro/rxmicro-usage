=== How It Works?

The RxMicro framework использует
{JDK-JAVA-COMPILER-DOC-ROOT-URL}javax/annotation/processing/Processor.html[Java annotation processor^], который генерирует стандартный код на основании `RxMicro annotations`.

Таким образом the RxMicro framework является фреймворком декларативного программирования.

Разработчик, используя the RxMicro framework концентрируется на написании бизнес логики микросервиса.
Затем с помощью `RxMicro annotations` конфигурирует желаемое стандартное поведение.
В момент компиляции проекта процессор аннотаций RxMicro генерирует вспомогательные классы.
Генерируемые классы содержат стандартную логику обеспечивающую работоспособность созданного микросервиса.
