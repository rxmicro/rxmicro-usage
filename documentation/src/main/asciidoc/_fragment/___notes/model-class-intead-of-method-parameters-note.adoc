[NOTE]
====
The RxMicro framework рекомендует для обработчиков запросов, зависящих от 3 и более
// tag::headers[]
HTTP headers
// end::headers[]
// tag::params[]
HTTP parameters
// end::params[]
// tag::path-variables[]
`path variables`
// end::path-variables[]
// tag::all[]
HTTP headers, HTTP parameters or `path variables`
// end::all[]
создавать отдельные классы модели запроса.

*При внедрении*
// tag::headers[]
*HTTP headers*
// end::headers[]
// tag::params[]
*HTTP parameters*
// end::params[]
// tag::path-variables[]
*`path variables`*
// end::path-variables[]
// tag::all[]
*HTTP заголовков, HTTP параметров или `path variables`*
// end::all[]
*непосредственно в обработчик, код становится трудночитаемым!*
====
