[NOTE]
====
In order to run `maven` commands in the terminal it is necessary to:

* Install `JDK` and `maven` on Your computer;
* Set the `JAVA_HOME` environment variable;
* Add the path to `$MAVEN_HOME/bin` folder to the `$PATH` environment variable.

_A detailed instruction on the `maven` installation for its further use in the terminal can be found at the following link: https://maven.apache.org/install.html[`Installing Apache Maven`^]._
====
