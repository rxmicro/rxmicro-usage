[[rest-controller-versioning-section]]
=== Versioning of REST Controllers

The RxMicro framework supports versioning of REST Controllers using two strategies:

* Versioning based on HTTP header analysis with the
{RX-MICRO-SRC-ROOT-BLOB-URL}rxmicro-http/src/main/java/io/rxmicro/http/HttpHeaders.java#L193[`Api-Version`^] name.
* Versioning based on URL Path fragment analysis.

<<<

==== Versioning based on HTTP header analysis

The RxMicro framework allows creating identical REST controllers that differ only in version:

[source,java]
----
include::{RX-MICRO-USAGE-ROOT-LOCAL-PATH}examples/rest-controller-versioning-header/src/main/java/io/rxmicro/examples/rest/controller/versioning/header/OldMicroService.java[tag=content,indent=0]
----
<1> REST controller of the old `v1` version, using the
{RX-MICRO-SRC-ROOT-BLOB-URL}rxmicro-rest/src/main/java/io/rxmicro/rest/Version.java#L84[`Version.Strategy.HEADER`^] strategy;

[source,java]
----
include::{RX-MICRO-USAGE-ROOT-LOCAL-PATH}examples/rest-controller-versioning-header/src/main/java/io/rxmicro/examples/rest/controller/versioning/header/NewMicroService.java[tag=content,indent=0]
----
<1> REST controller of the new `v2` version, using the
{RX-MICRO-SRC-ROOT-BLOB-URL}rxmicro-rest/src/main/java/io/rxmicro/rest/Version.java#L84[`Version.Strategy.HEADER`^] strategy;

include::../___notes/note-same-request-handlers-with-diff-versions.adoc[]

The correct selection of the appropriate REST controller handler can be checked with the following test:

[source,java]
----
include::{RX-MICRO-USAGE-ROOT-LOCAL-PATH}examples/rest-controller-versioning-header/src/test/java/io/rxmicro/examples/rest/controller/versioning/header/MicroServiceTest.java[tag=content,indent=0]
----
<1> The <<{testing}#testing-rest-based-microservice-tests-section,`@RxMicroRestBasedMicroServiceTest`>> annotation allows You to run several REST controllers in test mode.
<2> The test runs the `PATCH` request to a URL Path: `/`.
<3> To specify the handler version, the standard
{RX-MICRO-SRC-ROOT-BLOB-URL}rxmicro-http/src/main/java/io/rxmicro/http/HttpHeaders.java#L193[`Api-Version`^] HTTP header is used.
