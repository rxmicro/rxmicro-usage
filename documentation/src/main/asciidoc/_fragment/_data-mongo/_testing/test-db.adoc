==== Test Database

Для тестирования возможностей модуля `rxmicro.data.mongo`, которые описываются в данном разделе была создана тестовая база данных.
Тестовая база данных состоит из одной коллекции `account`, которая содержит 6 документов, описывающих учетные данные тестовых пользователей:

.Mongo Test Database
image::data-mongo/mongo-test-db.jpg[]
